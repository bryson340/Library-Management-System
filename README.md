# Library Management System

A complete Library Management System built with PHP (OOP) and MySQL.

## Features
* **Secure Login** (Password Hashing)
* **Admin Dashboard** (CRUD Operations)
* **Live Search** (AJAX & JSON)
* **Relational Database** (Users, Books, Issued Records)
* **Penalty Calculation** (Automatic fine for late returns)

## How to Run
1. Import `library_db.sql` into phpMyAdmin.
2. Update `Database.php` with your credentials.
3. Open the project in XAMPP.
